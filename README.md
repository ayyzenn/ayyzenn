### Who am I?

I'm **Saad Ahmad** — a student at **FAST NUCES** who builds things at the intersection of **DevOps** and **AI**.

I automate infrastructure with Ansible and Terraform, build CI/CD pipelines with Jenkins and Docker, and design multi-agent RAG systems that let you talk to your databases in plain English. All of it runs on my **Arch Linux + i3** setup. No cloud APIs. No shortcuts.

I also installed Gentoo from scratch once. On purpose.

### Things I've built

| Project | What it does |
| :--- | :--- |
| **Multi-Agent RAG System** | 3 AI agents (Router, Basic, Advanced) collaborate with ChromaDB, HyDE, and query decomposition |
| **Text-to-SQL Agent** | Ask questions in English, get answers from a database. Ollama + LangChain + SQLAlchemy |
| **Local RAG Knowledge Base** | Fully offline RAG. Ollama + ChromaDB + sentence-transformers. No internet needed |
| **Jenkins + Docker Pipeline** | GitHub → Jenkins → Docker build → container run. Fully automated CI/CD |
| **Ansible for Docker & K8s** | Playbooks that install Docker Engine and prepare Kubernetes nodes on Ubuntu servers |
| **Web Scraping API** | FastAPI + Flask backend with BeautifulSoup/Selenium, storing data in PostgreSQL + MongoDB |
| **Gentoo from Scratch** | Minimal CLI installer. Compiled everything. Because why not |
